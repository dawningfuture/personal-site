Inu

Live Preview

Repository

Inu is the product of the group project we completed for the Software Engineering Projects and Tools course I took in the Fall semester of 2014. It utilizes the Google Maps Javascript API to display dog-friendly establishments and public facilities (Inu is Japanese for "dog"). Bootstrap is used to provide a responsive framework, which I then turned into a sliding-sections SPA (Single Page Application). As a final component, one of my teammates created a blog for the site, which can be viewed on the live preview.
